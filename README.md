# rental-app-gui-csharp
RTA Rentals eshte nje aplikacion desktop i zhvilluar ne C# (Windows Forms) per menaxhimin e nje kompanie qiraje automjetesh. Aplikacioni perdor SQL Server per ruajtjen e te dhenave mbi automjetet dhe perdoruesit, ofron funksione login, signup, perditesim gjendjeje dhe shfaqje te dhenash live permes nje nderfaqeje te thjeshte dhe funksionale.
